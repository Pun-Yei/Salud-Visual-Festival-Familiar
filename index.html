Evaluación y Agudeza Visual

¡Cuidemos nuestros ojos desde pequeños!

¿Qué es la agudeza visual?

Es la capacidad que tenemos para ver claramente los objetos, letras o figuras a distintas distancias. Se mide con pruebas visuales para saber si necesitas lentes o algún tipo de apoyo.

Evaluación visual

La evaluación visual revisa cómo funciona tu vista. Puede detectar:

  • Miopía: dificultad para ver de lejos.
  • Hipermetropía: dificultad para ver de cerca.
  • Astigmatismo: visión borrosa o distorsionada.
  • Presbicia: pérdida de enfoque con la edad.
Partes del ojo humano
¿Cómo saber si alguien necesita una revisión visual?
  • Cierra o cubre un ojo frecuentemente.
  • Se tropieza con objetos fácilmente.
  • Se acerca mucho al leer o escribir.
  • Parpadea mucho o le cuesta leer.
  • No distingue letras o números con claridad.
¿Qué hacer si se detecta una dificultad visual?
  • Acudir a una clínica visual u hospital público para evaluación.
  • Solicitar apoyo con organizaciones que donan anteojos o hacen campañas.
  • Buscar seguimiento con oftalmología o rehabilitación visual si se requiere.
